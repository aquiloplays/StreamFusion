# StreamFusion 1.23.0

## Counter events in your feed

Your **Counters** (deaths, push-ups, whatever you track on aquilo.gg/counters)
now show up as their own rows in the events feed — however you bump them: the
dock, a Stream Deck, chat, or a plain link. Each change reads like
`Deaths → 12 (+1)` with its own violet badge.

- **Zero setup if you use Jukebox.** Counters reuse your Jukebox read key
  automatically — just flip on *Settings → Connections → Counters → Enable
  Counter events*. Not on Jukebox? Paste a counter feed URL from aquilo.gg/bot.
- **Read-only.** StreamFusion only *watches* your counters; it never changes
  them. Nothing is posted to chat and no OBS alert fires — it stays in the feed.
- Off by default; nothing changes until you enable it.
