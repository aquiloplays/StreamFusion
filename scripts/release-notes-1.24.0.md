# StreamFusion 1.24.0

## Penance — auto push-ups from stream events

Turn follows, subs, and hype trains into push-ups automatically. When it's
armed, StreamFusion adds to your **push-ups counter** (aquilo.gg/counters) as
events land — no Stream Deck press, no manual bump.

The tariff:

- **Follow → +10** — Twitch, Kick, and a YouTube subscribe each count on their
  own (all three = 30, not 10).
- **Twitch sub → +15 / +25 / +35** for Tier 1 / 2 / 3 (resubs too).
- **Kick sub → +20.**
- **YouTube membership → +15.**
- **Gift subs** count per gift, with a **cap per gift-bomb** (default 100, so a
  50-gift raid doesn't literally end you).
- **Hype train → +25 / +50 / +100** per level reached (levels 1–4 / 5–8 / 9+).

Set it up under **Settings → Connections → Penance**: enable it and paste your
push-ups counter's bump link from aquilo.gg/counters. Your existing
push-ups → lifetime counter link keeps the lifetime total in sync. Off by
default; nothing fires until you arm it.

## Also in this build
