# StreamFusion 1.17.1: settings search

A follow-up to the settings redesign.

## 🔍 New

- **Search settings**: a search box at the top of the settings sidebar
  filters the pages as you type, and it looks inside each page too, so
  typing "marker" finds the Stream page even though no tab is named that.
  Enter opens the first match, Esc clears.

## 🔧 Also

- New developer screenshot tooling for UI verification (scripts/main-shot.mjs).
