How to install
  1. Click Download Installer above. It'll save StreamFusion-Beta-Setup-*.exe (~74 MB).
  2. Run the installer. It's per-user (no admin / UAC prompt) and installs to %LOCALAPPDATA%\Programs\StreamFusion Beta\ — side-by-side with stable SF. Neither install will touch the other.
  3. Launch StreamFusion Beta from your Start menu. The icon is amber/orange (stable's is blue/teal), so you can tell them apart on the taskbar.
  4. On first launch you'll see the Tier 3 access gate. Click Sign in with Patreon, authorize in your browser, and the app unlocks as soon as Patreon confirms your tier. You only do this once.
⚠️ Windows SmartScreen may warn "Unknown publisher" the first time — the build is unsigned. Click More info → Run anyway. Stable SF has the same warning on first install; we're working on code-signing for both.
Set up silent auto-updates (optional, one-time)

Without this step, the beta works fine — it just won't automatically pull new beta versions when they're published. You'll have to come back here to download new builds.

To enable silent auto-update, you need a GitHub fine-grained PAT with read access to the private beta repo:

  1. Open https://github.com/settings/personal-access-tokens/new. Sign in if asked.
  2. Settings:
    • Token name: SF Beta auto-update
    • Expiration: 1 year (max allowed)
    • Repository access: Only select repositories → search for and pick aquiloplays/StreamFusion-beta
    • Permissions → Repository permissions:
      • Contents: Read-only
      • Metadata: Read-only (auto-added)
      • (leave everything else untouched)
  3. Click Generate token, copy it (starts with github_pat_).
  4. Close StreamFusion Beta, open PowerShell, paste: 
    # Replace github_pat_xxx with the token you just copied
$tok = "github_pat_xxx"
$path = "$env:APPDATA\streamfusion-beta\beta-updater-token.txt"
New-Item -ItemType Directory -Path (Split-Path $path) -Force | Out-Null
$tok | Out-File -NoNewline -Encoding ASCII $path
icacls $path /inheritance:r /grant:r "$($env:USERNAME):R" | Out-Null
Write-Host "✓ Token saved to $path" -ForegroundColor Green
  5. Relaunch StreamFusion Beta. From now on every new beta release installs silently on next quit — no action needed.
🔒 The token is stored locally on your machine only. It's scoped to only read this one repo — even if it leaked, the worst case is someone else could download the same beta installer you already have.

Uninstall

Start → Add or remove programs → search "StreamFusion Beta" → Uninstall. Stable StreamFusion is a separate entry and stays untouched.

Your beta settings live at %APPDATA%\streamfusion-beta\ — the uninstaller doesn't delete this folder by default so a fresh install keeps your config. Delete it manually if you want a clean slate.
